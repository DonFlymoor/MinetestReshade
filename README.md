# MinetestReshade
## To install:
1. Download and unpack this repo
2. Run `ReShade.4.9.1.Modded.exe`. If Minetest isn't detected, click browse and navigate to minetest.exe.
3. When it askes you what API minetest uses, select opengl.
4.Click install, and chose the default settings (or chose whatever shader packs you want to play around with)
5. If that succeded, close the shader installer.

  __A__. If you want to play around wih shaders on your own, open minetest and hit the Home key. There will be a short tutorial which will teach you how to use it.

  __B__. If you just want fancy effects without spending a long time tweaking settings, copy `reshade-shaders`, `ReShade.ini`, and `ReShadePreset.ini` to the same folder that minetest.exe is in.
6. Lauch minetest.

7. Profit!

To edit the settings in the game, you will have to press escape before pressing home, or your mouse cursor will stick in the middle of the screen.
